Class Version

Vers::Version — Data structure to store a version number.

Synopsis

// In header: <Versionning/Version.h>


class Version {
public:
  // construct/copy/destruct
  Version();
  Version(unsigned short, unsigned short, unsigned short, unsigned short);
  Version(const std::string &);

  // public member functions
  unsigned short getMajor() const;
  unsigned short getMinor() const;
  unsigned short getPatch() const;
  unsigned short getTweak() const;
  std::string toString() const;
  void setMajor(unsigned short);
  void setMinor(unsigned short);
  void setPatch(unsigned short);
  void setTweak(unsigned short);
  void set(unsigned short, unsigned short, unsigned short, unsigned short);
  void set(const std::string &);
};

Description

Version public construct/copy/destruct

  1. Version();
    Default constructor.

    Version is initialized to v0.0.0.0 

  2. Version(unsigned short major, unsigned short minor, unsigned short patch, 
        unsigned short tweak);
    Constructor taking 4 numbers as parameters.

    Parameters:

    major

    1st version number

    minor

    2nd version number

    patch

    3rd version number

    tweak

    4th version number 

  3. Version(const std::string & version);
    Constructor from a string.

    See Also:

    Version::set(const std::string&)

    Parameters:

    version

    a string representing a version number. This string must be well formed.

    Throws:

    		 VersionException

Version public member functions

  1. unsigned short getMajor() const;
    Get 1st version number. 
  2. unsigned short getMinor() const;
    Get 2nd version number. 
  3. unsigned short getPatch() const;
    Get 3rd version number. 
  4. unsigned short getTweak() const;

    Set 4th version number.

         unsigned short tweak);
    Set complete version number from 4 numbers. 
  11. void set(const std::string & version);
    Set complete version number from a string.

    The given string must be well formed. It must be composed of 1 to 4 unsigned numbers separated by a dot '.' or comma ','. If the string is bad formed, an exception is thrown. This method provides the strong exception-safety guarantee.

    Parameters:

    a string containing the version number
