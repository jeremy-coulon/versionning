Function operator>=

Vers::operator>= — Greater-or-equal operator of 2 version numbers.

Synopsis

// In header: <Versionning/Version.h>


bool operator>=(const Version & v1, const Version & v2);

Description

Parameters:

v1

First version number

v2

Second version number

Returns:

True if the first version number is greater than or equal to the second version number. False otherwise.
